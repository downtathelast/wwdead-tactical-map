# WWDead Scripts – Public Repository

Welcome! This repository contains **final, ready-to-use WWDead scripts** for players.

---

## 📦 Available Scripts

Each script folder contains the built userscript in `/dist`:


SCRIPT_NAME/
└─ dist/ ← Userscript file(s) ready to install


**Example:**

TacticalMap/
└─ dist/TacticalMap.user.js
NameColorizer/
└─ dist/NameColorizer.user.js


---

## ⚡ Installation

1. Install a userscript manager in your browser (Tampermonkey, Greasemonkey, or Violentmonkey).  
2. Open the `.user.js` file from `/dist` in your browser.  
3. Confirm the install.  
4. Reload Urban Dead — your script is active!

> Only `/dist/*.user.js` files are needed.
